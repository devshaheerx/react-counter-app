# React Counter App

A sleek counter app built with React and Tailwind CSS, featuring a glassmorphism card UI over a looping video background.

## Features
- Increment, decrement, and reset counter
- Color-coded count display (green / red / neutral)
- Glassmorphism card with backdrop blur
- Animated buttons with hover/active states
- Video background with overlay

## Tech Stack
- React 18
- Tailwind CSS
- Vite
- Lucide React (icons)

## Getting Started
```bash
npm install
npm run dev
```
## Build for Production
```bash
npm run build
```
